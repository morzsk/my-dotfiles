# My NixOS Dotfile

My dotfiles
```md
.
├── flake.nix               (My flake)
├── flakes                  
│   ├── default.nix         
│   ├── home-manager.nix    (Home Manager Flakes)
│   └── nixos.nix           (NixOS Flakes)
├── hosts                   
│   ├── laptop              (Laptop imports)
│   └── pc                  (Pc imports)
└── modules
    ├── system              (System level configs)
    └── user                (User level configs)
```

My Setup
- OS -- NixOS (Flakes + Home Manager)
- WM -- Hyprland
- Display Manager -- Ly
- Shell -- Zsh
- Terminal -- Kitty
- Editor -- NvChad (Neovim)
- Browser -- Brave
- File Manager -- Yazi
- Launcher -- Rofi
- Theme -- Gruvbox dark via Stylix
- Font -- JetBrainsMono Nerd Font
- Cursor -- Capitaine Cursors (Gruvbox)
- Fetch -- fastfetch
- Notes -- Obsidian
